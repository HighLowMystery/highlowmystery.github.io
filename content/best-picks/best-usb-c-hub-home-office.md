## Mac vs Windows Compatibility: What You Need to Know

Most USB-C hubs work on both platforms — the ports are the same. The differences show up in:

- **DisplayLink vs native Alt-Mode video**: Macs don't support DisplayLink natively post-Apple Silicon. If a hub uses DisplayLink for video, you need the DisplayLink driver installed on Mac. Without it, video output won't work.
- **Power delivery passthrough**: MacBooks are picky about PD specs. Look for 87W+ PD passthrough if you're on a 14/16-inch MBP.
- **Thunderbolt vs USB 3.2**: Thunderbolt hubs cost more but deliver full bandwidth for everything simultaneously. USB 3.2 hubs are fine unless you're running 4K@60Hz + SSD + charging at the same time.

{{< callout type="tip" >}}}
**Mac users:** Check if video output uses Alt-Mode (no driver needed) or DisplayLink (driver required). For Apple Silicon Macs, Alt-Mode is almost always simpler.

## Buying Guide: How to Pick the Right Hub

**First: check your laptop's USB-C port spec**
- USB 3.2 only → any hub works, but limit simultaneous 4K + high-speed storage
- Thunderbolt 3/4 → unlocks full-bandwidth docks like CalDigit

**Ports you actually need for WFH:**
- HDMI (for monitor) — check 4K@60Hz if you have a 4K monitor
- USB-A ×2 minimum (keyboard + mouse or dongle)
- Ethernet (Wi-Fi is unreliable for video calls)
- USB-C PD passthrough (keep laptop charged)
- SD card reader (optional but useful)

**Red flags:**
- No listed PD wattage — means it probably slow-charges or doesn't charge at all
- "Supports 4K" without specifying Hz — usually means 30Hz
- No mention of Mac compatibility if you're on Apple Silicon
