# Work From Home Pro — Affiliate Site

Affiliate site in the Remote Work / Ergonomics / Home Office niche.
Built with Hugo. Hosted on GitHub Pages.

## Local Dev

```bash
hugo server --buildDrafts
```

## Build

```bash
hugo --minify
```

## Deploy

Push to `main` → GitHub Actions auto-deploys to GitHub Pages.

## Structure

- `content/best-picks/` — "Best X for Y" posts (primary traffic)
- `content/comparisons/` — "X vs Y" posts (high conversion)
- `content/how-to/` — "How to X" posts (authority + email capture)

See `SITE-PLAN.md` for full strategy.
